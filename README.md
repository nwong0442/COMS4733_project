# Ball-Pushing Robotic Arm in Maze

This project simulates a MuJoCo-based robotic arm navigating a maze using Jacobian-based control.

## Requirements
- Python 3.x
- MuJoCo
- Jupyter Notebook
- numpy
- pygames

## Running the Simulation
The main simulation file is `project.ipynb`.

1. Open the notebook in Jupyter.
2. Run all cells to start the simulation.
